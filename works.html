職業訓練校で制作した作品

Concept

オーガニック商品を専門的に扱うお店。ナチュラルでお洒落な雰囲気希望。ECサイト、エコバッグ等に使いたい。

Point

店名を周知していただく必要があると考え、ロゴを大胆に使用。丸みのある筆記体を使用し、女性向けの印象に。年齢層に合わせ写真、文字のトーンは抑え、落ち着いた、優しい印象になるようにした。

Tool

llustrator、Photoshop / 構想：30分、制作:３時間　

  • 採用案

    まとまりがあるデザインが写真に埋もれることなくパッと目をひく。コモレビのイメージが一番伝わりやすい。

  • その他の候補

    線で表現しているのでロゴとしてのインパクトが弱い。写真と一緒に使わずロゴのみ(カード、エコバッグ)の使用に向いている。

  • 縁取りがあり、はっきりしているのでナチュラルというよりはスタイリッシュな雰囲気の方が合っている。

Tool

llustrator / 構想：各30分 / 制作：各3時間　

グループ制作

仮想のブランド苺　ランディングページ(コーディング済み)

WEB版はここをClick!!

Concept

苺と共に人生を歩んできた匠が、愛娘のように大切に育てた朝摘み苺の「いち娘」。30代独身女性がターゲットで女子会や贈り物として購入することを想定。エレガントなイメージを前面に出したデザインに。

Point

５人チームで作業を分担。シナリオシートで具体的なペルソナを設定し、完成イメージを全員で共有してから作業を進めた。デザイナー、コーダー、画像処理、ライターをそれぞれが担当し、私の担当範囲は画像処理とロゴの提案。Google Driveを使用し、データーを共有しながら作業を進めた。

Tool

Illustrator、Photoshop、XD、Dreamweaver

Concept

おやつ感覚で毎日飲んで体質改善を狙う乳酸菌サプリ。30歳以上の子持ちの女性向け。商品の特徴や悩みの解決策として興味をひき、クリックしてもらう。リターゲティング広告。送料無料と記載して欲しい。

Point

同じ色、三角の装飾を使い、商品のイメージに近づけることで印象づけることを狙った。クレヨン風の線、枠に丸みを持たせて優しい印象になるようにした。色が喧嘩しないよう黄色の不透明度を下げることで見やすくした。

Tool

llustrator、Photoshop 構想：1時間 / 制作：5時間 / 写真：依頼者からの提供

Concept

春から夏に向けて販売する、沖縄旅行パッケージプランへの誘導バナー。20後半〜40代の独身男女をターゲットに作成。料金プランはやや高めの設定だがホテルのランクが高い等、非日常を味わえるプランになっている。

Tool

Photoshop / 構想：２時間/制作時：２時間/写真：自分で撮影

Concept

Tool

Tool

llustrator　構想：2時間 制作時間　3時間
